# Basic Express Website

**Initials Project**

```
npm install -g express
```

```
npm init
```

***Package.json***

```
{
  "name": "express_website",
  "version": "1.0.0",
  "description": "Simple website",
  "main": "app.js",
  "scripts": {
    "test": "echo \"Error: no test specified\" && exit 1"
  },
  "author": "author",
  "license": "ISC",
  "dependencies": {
  	"body-parser": "*",
  	"express": "*",
  	"jade": "*",
  	"nodemailer": "*"
  }
}
```

```
npm install
```

```
node app
```
